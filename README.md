# FowardNotification
## 内容
特定のアカウント（UTOL）から届いた通知の不要な行を削除して自分宛に転送するスクリプト。  
元の通知はラベルをつけられてゴミ箱へ移動される。
## ツール
GAS, Gmail
## 使い方
### GAS
1. 送信元のアカウント、転送先のアカウントを設定
2. 削除したい文言を設定
3. 元の通知につけるラベルを設定
4. 転送時の件名と送信者名を設定
5. トリガーを設定（時間主導型・10分おき）
### Gmail
1. 送信元のアカウントでフィルタを作成
2. 処理を「既読にする」に設定
## 作成理由
特定のアカウント（大学の学務システム UTOL）から定期的に届くメールに毎回同じ長文ヘッダーが含まれており、スマホの通知画面で内容を把握できなかったため。  
このスクリプトによって、スマホでも通知を見ればどの授業で何の更新があったかわかるようになった。
## 認識している問題点
- 新しいメールが過去のメールのスレッドに送信される場合、過去のメールが再度転送されてしまう。
